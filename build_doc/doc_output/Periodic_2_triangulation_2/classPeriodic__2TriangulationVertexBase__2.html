2D Periodic Triangulations Reference » Concepts

#include <Concepts/Periodic_2TriangulationVertexBase_2.h>

Definition

A refinement of the concept TriangulationVertexBase_2 which adds an API for offset.

At the base level of 2D-triangulations (see Section Section_2D_Triangulations_Software_Design), a vertex provides access to one of its incident faces through a handle.

The storage of the offset is only needed when a triangulation is copied.
